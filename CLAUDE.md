# PCCP Java 학습 프로젝트

## 목표
- PCCP Lv.3 이상 취득 가능한 코딩 역량 확보
- 언어: Java
- 플랫폼: 프로그래머스

## 학습 플랜 (12주)
- 노션 원본: 헤밀레라이프랩 / PCCP Java 학습 플랜 (12주)
- 시작일: 2026-04-01 (수요일)
- 방식: 매일 1문제 (30분~1시간), 30분 고민 → 못 풀면 풀이 확인 → 다음 날 복습

### Phase 1: 기초 체력 (1~3주차) - Lv.1
- 1주차: 문자열 · 배열 · 연결리스트 (번갈아 풀기)
- 2주차: 조건문 · 반복문 · 함수
- 3주차: 해시(HashMap) 기초

### Phase 2: 핵심 자료구조 · 알고리즘 (4~7주차) - Lv.2
- 4주차: 정렬 · 그리디
- 5주차: 스택 · 큐 · 덱
- 6주차: DFS · BFS
- 7주차: 이진 탐색

### Phase 3: 심화 알고리즘 (8~10주차) - Lv.2~3
- 8주차: 그래프 · 트리
- 9주차: 힙(PriorityQueue)
- 10주차: DP (동적 프로그래밍)

### Phase 4: 실전 모의고사 (11~12주차)
- 11주차: PCCP 기출 풀이
- 12주차: 약점 보완 + 모의 시험

## 프로젝트 구조
- `src/weekN/` — N주차 풀이 코드
- Claude가 문제 출제 + 코드 리뷰, 주 1~2회는 프로그래머스 실전 연습 병행

## 진행 규칙
- 문제 파일명: `Day{주차}_{YYMMDD}_{번호}.java` 형식 (예: `Day1_260401_1.java`)
- 패키지: `week1`, `week2`, ... 형식
- import는 `import java.util.*;` 와일드카드 사용
- 문제 주석에 힌트(사용할 API, 접근 방식 등)를 포함할 것 (사용자가 빼달라 할 때까지 유지)

## 문제 출제 형식
- Scanner 입출력 방식이 아닌, 프로그래머스 스타일로 출제할 것
- `solution()` 메서드에 매개변수로 입력을 받고, 결과를 return하는 형태
- `main()`에는 테스트 케이스를 넣어서 PASS/FAIL로 검증
- 사용자는 핵심 알고리즘 로직만 작성하면 되도록 구성
